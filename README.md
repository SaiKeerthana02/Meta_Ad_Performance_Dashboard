# 📊 Meta Ad Performance Dashboard

## Overview
Power BI dashboard for tracking Meta Ads performance across Facebook and Instagram.

## 📈 KPIs Tracked
- Impressions: 216.0K
- Clicks: 25.4K
- Shares: 1.3K
- Comments: 2.6K
- Purchases: 1.3K
- Engagements: 29.3K
- CTR: 11.76%
- Engagement Rate: 13.56%
- Conversion Rate: 5.21%
- Purchase Rate: 0.61%
- Total Budget: $2.5M
- Avg. Budget per Campaign: $50.7K

## ✨ Features
- Facebook & Instagram separate tabs
- Dynamic measure selector (Engagements, Clicks, etc.)
- Campaigns Name filter
- Target Interest filter
- Engagements by Gender (Male 22%, Female 43%, All 35%)
- Engagements by Age group
- Engagements by Country (World Map)
- Weekly Engagements Trend
- Hourly Engagements Trend
- Monthly Analysis with Calendar view
- Analysis by Ad Type (Video, Stories, Image, Carousel)

## 🛠️ Tool Used
- Power BI Desktop
